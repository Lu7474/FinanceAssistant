# FinanceAssistant

Веб-приложение для личного учёта финансов на Django. Позволяет отслеживать доходы и расходы по категориям с системой авторизации пользователей.

## Функционал

- **Авторизация** — регистрация, вход и выход из системы
- **Категории** — создание и удаление категорий для доходов/расходов
- **Доходы** — добавление, просмотр и удаление записей о доходах
- **Расходы** — добавление, просмотр и удаление записей о расходах
- **Итоги** — автоматический подсчёт общей суммы доходов и расходов
- **Детализация** — просмотр транзакций по конкретной категории
- **Изоляция данных** — каждый пользователь видит только свои данные

## Технологии

- Python 3.x
- Django 4.2
- SQLite3
- Bootstrap 5.3
- HTML5 / CSS

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Lu7474/FinanceAssistant.git
   cd FinanceAssistant
   ```

2. Создайте и активируйте виртуальное окружение:
   ```bash
   python -m venv env

   # Windows:
   env\Scripts\activate
   # Linux/MacOS:
   source env/bin/activate
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Примените миграции:
   ```bash
   python manage.py migrate
   ```

5. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

6. Откройте в браузере: http://127.0.0.1:8000

## Остановка сервера

Используйте комбинацию клавиш `Ctrl+C` в терминале.

## Использование

1. Зарегистрируйтесь или войдите в систему
2. Создайте категории для классификации транзакций
3. Добавляйте доходы и расходы, выбирая соответствующие категории
4. Просматривайте итоговые суммы на главной странице
5. Кликните на категорию для просмотра связанных транзакций
